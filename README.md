- Node

## Run app "Development" ⚙️

```
1.- npm install or yarn install
2.- npm run dev or yarn dev
```

## Run test 🧪

```
1.- npm run test
```

## Run app "Production" 🚀

```
1.- npm install or yarn install
2.- npm run build or yarn build
3.- npm run start or yarn start
```
